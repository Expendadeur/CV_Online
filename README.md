CV Studio
L'application CV Studio est maintenant prête. Elle permet de créer un CV professionnel complet dans une interface moderne et fluide, tout en étant contenue dans un seul fichier 
index.html
.

Fonctionnalités Implémentées
1. Page d'Accueil (Landing Page)
Design épuré avec un bouton d'appel à l'action "Créer mon CV".
Animations fluides à l'entrée.
2. Formulaire de Saisie en 9 Étapes
Infos Personnelles : Photo de profil, Nom, Titre, Contact.
À Propos : Biographie professionnelle.
Formation : Ajout dynamique de diplômes.
Expérience : Ajout dynamique de postes occupés.
Compétences : Gestion de compétences techniques (niveaux) et soft skills.
Projets : Mise en avant de réalisations.
Langues : Niveaux certifiés.
Centres d'intérêt : Personnalisation finale.
Signature Électronique : Outil de dessin intégré pour signer votre CV.
3. Aperçu en Temps Réel
Le panneau de droite affiche une simulation A4.
Chaque frappe clavier met à jour le rendu instantanément.
4. Personnalisation & Export
Thèmes : 6 couleurs professionnelles (Bleu, Vert, Rouge, Violet, Orange, Noir).
Layouts : 2 modèles (Moderne avec sidebar ou Classique empilé).
Export PDF : Génération de fichier PDF haute qualité.
Export HTML : Téléchargement du code source du CV seul.
Sauvegarde : Les données sont persistées automatiquement dans le navigateur (localStorage).
